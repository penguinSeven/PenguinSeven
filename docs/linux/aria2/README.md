# aria2

## 安装

- debian安装

```bash
$ sudo apt-get install aria2
```

## 常用命令行
```bash
# 多个进程下载
$ aria2c -x2 http://xx.xx.com/123.jpg
```

## 启动

- 允许rpc 

```bash
$ aria2c --enable-rpc --rpc-listen-all
```
