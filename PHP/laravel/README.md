       |   ├─ Events --------------------------------------- 应用公共模块
       |   ├─ Exceptions --------------------------------------- 应用公共模块

       |   ├─ Listeners --------------------------------------- 应用公共模块

       |   ├─ Providers --------------------------------------- 应用公共模块
       ├─ bootstrap --------------------------------- 应用目录
       ├─ config --------------------------------- 应用目录
       ├─ database --------------------------------- 应用目录
       ├─ public --------------------------------- 应用目录
       ├─ resources --------------------------------- 应用目录
       ├─ routes --------------------------------- 应用目录

       |   ├─ template -------------------------------- 模板文件目录

       |   ├─ Applications --------------------------- socket业务逻辑目录

       |   ├─ framework.php -------------------------- 核心框架文件

       |   ├─ getSwitch.php -------------------------- 指令处理文件
       |   ├─ rasterize.js --------------------------- phantomjs文件
       |   └─ mysql.php ------------------------------ 数据库操作核心类
